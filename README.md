
# Portfolio Professionnel - ADJOLOU Dondah Chaminade

Ce projet est un portfolio moderne, professionnel et haute performance conçu pour présenter les compétences en développement Web & Mobile de Dondah Chaminade.

## Caractéristiques
- **Design :** Glassmorphisme, thème Bleu Bordeaux, animations slivers.
- **Framework :** React + Tailwind CSS + Framer Motion.
- **Contenu :** Expériences chez Efficorpe et Axone Digital, Éducation, Engagement communautaire.
- **Responsive :** Optimisé pour mobile et desktop.

## Installation
1. Installez les dépendances avec `npm install`.
2. Lancez le serveur de développement avec `npm run dev`.
